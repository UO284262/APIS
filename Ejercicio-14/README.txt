Mi aplicación es un sencillo reproductor de música.
Utilizando la API FILE nos permite escuchar cualquiera de las canciones que están ubicadas en la carpeta Musica.

Posteriormente utiliza la API Web Audio para implementar el reproductor. En este caso se han añadido dos modificadores
concretos. El de volumen y el de estereo. Manejando el estereo en que auricular se escucha más fuerte el audio.

Por últimmo, se ha añadido la API Web Visibility para disminuir el volumen de la cancion al ocultarse la pestaña y 
devolverlo a su nivel original al mostrarse.
